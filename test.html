📖 Markdown Viewer テストページ

このページは拡張機能のテスト用です。アイコンをクリックしてマークダウン表示を試してください。

🔧 テスト手順

  1. Chrome拡張機能のアイコンをクリック
  2. ページがマークダウン形式に変換されることを確認
  3. 再度アイコンをクリックして元の表示に戻ることを確認

サンプルコンテンツ

これは拡張機能のテスト用ページです。以下のような様々なコンテンツ要素を含んでいます：

  • 見出し（H1, H2, H3）
  • 段落テキスト
  • リスト項目
  • コードブロック
  • テーブル

コード例

// マークダウン変換の例 const markdown = toMarkdown(extractedContent); console.log(markdown);
